# Variables binarias (dicotómicas)

Sean X e Y dos variables dicotómicas (binarias) definidas sobre una población
finita Ω.

Cada variable puede tomar exactamente dos valores posibles, que denotamos como:
- X ∈ {0, 1}
- Y ∈ {0, 1}

En esta etapa no se asume ningún modelo probabilístico. El análisis es
estrictamente combinatorio y se basa únicamente en las ocurrencias conjuntas
de los posibles valores de X e Y.

Variables binarias de este tipo aparecen de forma natural en múltiples
contextos, por ejemplo:
- Presencia / ausencia de una característica
- Respuestas sí / no
- Resultados positivos / negativos
- Indicadores verdadero / falso

La asociación entre X e Y se caracterizará exclusivamente a través de su
estructura conjunta de contingencia.
