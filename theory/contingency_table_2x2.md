# Tabla de contingencia 2×2

Sean X e Y dos variables dicotómicas definidas sobre una población finita Ω.
La relación conjunta entre X e Y puede representarse mediante una tabla de
contingencia 2×2, caracterizada por cuatro valores no negativos:

- a: número de elementos tales que X = 1 e Y = 1
- b: número de elementos tales que X = 1 e Y = 0
- c: número de elementos tales que X = 0 e Y = 1
- d: número de elementos tales que X = 0 e Y = 0

Estos valores satisfacen la condición:

a + b + c + d = n,

donde n = |Ω| es el tamaño total de la población.

La tabla de contingencia puede representarse de la siguiente forma:

|        | Y = 1 | Y = 0 |
|--------|-------|-------|
| X = 1  |   a   |   b   |
| X = 0  |   c   |   d   |

Esta representación es puramente estructural y no presupone ningún modelo
probabilístico ni estadístico. Todas las medidas de asociación consideradas
en este trabajo se definen como funciones de los valores (a, b, c, d).

En particular, el análisis experimental se realizará exclusivamente sobre
el espacio de todas las tablas de contingencia 2×2 válidas, lo que permite
un estudio controlado y exhaustivo del comportamiento de las medidas de
asociación.
